# 文档总目录

## 摘要
本项目是一个基于 Astro 的个人博客站点。当前文档体系用于沉淀站点结构、构建方式、首页作者信息入口和常见维护点，后续修改请优先从本索引进入，避免重复全仓搜索。

## 查阅顺序
1. 先看 `docs/01_项目架构与模块职责.md` 了解页面、布局、组件和内容源的关系。
2. 再看 `docs/02_开发环境与构建部署.md` 确认宿主机、开发命令和构建约束。
3. 若涉及首页作者区、浏览器标题或默认站点描述，直接看 `docs/06_首页作者信息与站点标题.md`。
4. 若涉及 3D 展厅模型分组、模型路径或模型说明，直接看 `docs/07_3D展厅与模型分组.md`。
5. 若涉及全站右下角回到顶部按钮，直接看 `docs/08_回到顶部按钮.md`。
6. 若只想复盘具体改动过程，再查看 `docs/plans/` 下对应的计划文档。
