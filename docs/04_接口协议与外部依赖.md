# 接口协议与外部依赖

## 摘要
本项目的核心外部依赖集中在 Astro 生态、Markdown 渲染、评论系统和前端资源加载。首页作者信息和浏览器标题的调整不新增依赖，但会直接使用布局层和组件层已有的站点元信息输出链路。

## 依赖清单表
| 依赖项 | 类型 | 用途 | 所在位置 | 使用入口 | 备注 |
| --- | --- | --- | --- | --- | --- |
| `astro` | 框架 | 站点构建、路由和组件渲染 | `package.json` | 全站 | 核心依赖 |
| `@astrojs/mdx` | Astro 集成 | 支持 MDX 内容 | `package.json` | 内容层 | 文章系统相关 |
| `@astrojs/rss` | Astro 集成 | 生成 RSS | `package.json` | 站点输出 | 供订阅使用 |
| `@astrojs/node` | Astro 适配器 | 开发态 API 路由支持 | `package.json` | `astro.config.mjs` | 当前仅开发模式启用 |
| `axios` | JS 库 | 网络请求封装 | `package.json` | API/工具逻辑 | 本次改动未涉及 |
| `marked` | JS 库 | Markdown 解析 | `package.json` | 编辑器或内容处理 | 本次改动未涉及 |
| `dompurify` | JS 库 | HTML 清洗 | `package.json` | 内容安全处理 | 本次改动未涉及 |
| `giscus` | 外部服务 SDK | 评论系统接入 | `package.json` | 评论组件 | 依赖 GitHub Discussions |

## 外部资源与协议
| 外部资源 | 类型 | 用途 | 入口位置 | 备注 |
| --- | --- | --- | --- | --- |

## 与本次改动的关系
1. 本次仅改首页文案、页签标题和默认站点描述，没有新增第三方依赖。
2. 浏览器页签标题依赖 `Layout.astro` 的 `<title>` 输出链路，不涉及外部服务调用。
3. 作者介绍与标签完全位于 `AuthorProfile.astro`，不依赖 API 返回数据。
